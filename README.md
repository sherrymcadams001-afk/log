# Cedrick Logging - Tech-Stewardship Web Portfolio

A modern web portfolio for **Cedrick Logging** (Owner: C. Thomas), merging Corporate ESG authority with Modern Mass-Timber aesthetics.

## Design Philosophy

This portfolio implements a "Tech-Stewardship" approach that combines:
- **Corporate ESG Authority**: Trust indicators, certifications, carbon tracking
- **Modern Mass-Timber Aesthetics**: Clean, sophisticated design without rustic elements

## Features

### Visual Design
- **Tech-Timber Palette**: White (#FFFFFF), Pine (#C4956A), Black (#1A1A1A)
- No rustic or muddy greens - clean, modern aesthetic
- Responsive design for all devices

### UX Features
- **Immediate Split-Path Navigation**: Clear paths for Landowners vs Buyers
- Smooth scroll navigation
- Interactive elements with hover states

### Technical Features
- **GIS/LiDAR Mapping Section**: Showcases precision forest mapping capabilities
- **Carbon Calculator**: Interactive tool to estimate carbon sequestration value
- **Drone Video Background**: Hero section designed for video integration
- **Trust Indicators**: Key statistics and certifications

## File Structure

```
/
```

## Usage

Simply open `index.html` in a web browser to view the portfolio.

## Conversion Logic

The portfolio implements strategic conversion paths:

1. **Landowner Path**: Assessment → GIS Mapping → Carbon Credits → Consultation
2. **Buyer Path**: Inventory → Certification Docs → Pricing → Consultation

Both paths converge at consultation requests, maximizing lead generation.
